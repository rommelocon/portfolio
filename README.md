## ğŸš€ Deploying to GitHub Pages

1. Install `gh-pages`:

```bash
npm install gh-pages --save-dev
```

2. Set `base` in `vite.config.ts`:

```ts
base: '/portfolio/',
```

3. Add deploy script in `package.json`:

```json
"deploy": "vite build && gh-pages -d dist"
```

4. Deploy:

```bash
npm run deploy
```
